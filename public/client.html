Interactive Classroom

Problems faced

Adding embedded ppt to webpage, made tracking events difficult.

Html event handlers cannot be used for iframe tags.

Changing to fullscreen mode, changes current window.

Editing over embedded ppts not possible due to cross origin domain.

Ajax and Http request too complex to handle dynamic events.

Proposed model

Make ppt using Markup(HTML) Language.

Added a layer of canvas over html and manipulated it with javascript.

Communicate with clients using WebSockets.
